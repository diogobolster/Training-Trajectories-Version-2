# Next Experiment: Generalization Test

## Goal

Reduce the largest reviewer risk: the current evidence comes from one Bentheimer trajectory set generated by one approximate flow pipeline. The next experiment should test whether validation-driven mixture selection survives a new physical condition.

## Recommended Experiment

Run the same TTA-v2 benchmark on a **second Peclet condition** first, then on a **second geometry**, then on a **higher-fidelity velocity field**.

Why Peclet first:

- It can reuse the existing Bentheimer geometry and pressure/velocity field.
- It only requires regenerating particle trajectories with a changed diffusivity.
- It directly tests finite-Peclet behavior, which was a core motivation of the original 2019 paper.
- It is faster than finding, downloading, segmenting, and validating a new rock.

Why geometry second:

- It is the stronger generalization claim.
- It addresses whether learned/mixture selection is tied to Bentheimer pore structure.
- It is closer to the long-term TTA-v2 goal of geometry-conditioned transport.

## Experiment A: Second Peclet Condition

### Design

Use the same downsampled Bentheimer 6 micrometer geometry and approximate velocity field, but regenerate trajectories at a different diffusivity.

Current main value:

```text
diffusivity = 0.001
```

Recommended new values:

```text
low diffusion / high Peclet:  diffusivity = 0.0003
high diffusion / low Peclet:  diffusivity = 0.003
```

At minimum, run one of these. Ideally run both.

### Status

The high-Peclet and low-Peclet cases have now been run:

```text
diffusivity = 0.0003
diffusivity = 0.003
```

See [run_008_peclet_generalization.md](run_008_peclet_generalization.md) and [run_009_peclet_sweep.md](run_009_peclet_sweep.md).

### Expected Scientific Value

High Peclet should increase velocity-memory importance and may favor Gaussian/Bayes or learned context. Lower Peclet should increase diffusive smoothing and may reduce the need for long-memory transition structure. If selected mixture weights shift consistently with diffusivity, that becomes a strong mechanistic result.

## Experiment B: Second Geometry

### Candidate 1: Zenodo Second Bentheimer Subvolume

Use another subvolume from the same Zenodo multi-resolution Bentheimer dataset.

Pros:

- Same file format and pipeline.
- Fastest second-geometry test.
- Tests spatial variability within Bentheimer.

Cons:

- Not a new rock type.

### Status

```text

```

```text
gaussian_bayes               mean_obj 255.40   mean_rank 1.75   wins 2
hybrid                       mean_obj 268.50   mean_rank 3.25   wins 1
```

Mean selected weights:


Interpretation: the original physics-informed kernel becomes the strongest fixed sampler on Core2, but validation still selects mixed mechanisms. This strengthens the no-universal-winner framing.

## Metrics to Compare Across Conditions

For each new condition, report:

- best sampler by outer-split mean objective,
- mean selected mixture weights,
- whether pooled validation mixture beats Gaussian/Bayes and hybrid,
- objective-weight sensitivity heatmap,
- shift in pair/dilution/reaction performance.

## Decision Criteria

The next experiment is successful if at least one of the following holds:

1. The validation-selected mixture remains competitive with Gaussian/Bayes on the new condition.
2. The selected mixture weights shift in a physically interpretable way with Peclet number or geometry.
3. The result reveals that Gaussian/Bayes is dominant under a specific condition, strengthening the paper's no-universal-winner claim.
