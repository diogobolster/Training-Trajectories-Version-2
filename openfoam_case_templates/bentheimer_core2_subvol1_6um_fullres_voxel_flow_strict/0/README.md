# Full-Resolution Initial Field Note

The full-resolution `0/U` field is omitted from the GitHub package because the copied OpenFOAM field file is too large for normal GitHub storage. The full field is part of the large OpenFOAM artifact listed in `DATA_MANIFEST.md` and should be deposited in the DOI-bearing archive with the mesh and solved time directories.

The solver controls, pressure boundary file, transport properties, turbulence settings, and case metadata are retained here so the case configuration remains inspectable.
